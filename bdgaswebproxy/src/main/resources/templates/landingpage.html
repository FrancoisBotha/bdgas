Audit Analytics Cloud

An implimentation of the Open Source Big Data Enabled Generalized Audit Software (BDGAS) project. Perform standard audit analytics on Big Data sets on a cluster of computers - greatly enhancing the speed of processing.

Includes industry standard security practices such as encryption of data in transit and at rest, OpendID Connect and Spring Security.

Extend functionality by creating custom audit test plugins.
